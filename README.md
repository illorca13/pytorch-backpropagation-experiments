# pytorch-backpropagation-experiments
Exploring computational graphs, gradient flow, and backpropagation using PyTorch.

# Understanding Backpropagation in PyTorch
This project explores how PyTorch constructs computational graphs
and computes gradients using automatic differentiation.

## Objectives
- Understand tensor structure and memory layout
- Explore PyTorch autograd
- Verify gradients manually
- Investigate vanishing and exploding gradients

## Tools
Python
PyTorch
Jupyter Notebook

## Key Concepts
Computational graphs
Gradient flow
Backpropagation
Automatic differentiation
